\chapter*{ЗАКЛЮЧЕНИЕ}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

Цель, которая была поставлена в начале работы, была достигнута:
разработан HTTP-сервер для отдачи статического
содержимого, построенного на архитектуре пула потоков с мультиплексированием сетевых
соединений с использованием системного вызова \texttt{poll()}.

Решены все поставленные задачи:

\begin{itemize}
	\item[---] Предусмотрена поддержка запросов GET и HEAD, поддержка статусов 200, 403, 404.
	\item[---] Предусмотрена возможность ответа сервера на неподдерживаемые запросы статусом 405.
	\item[---] Обеспечена корректная передача файлов размером до 128 Мбайт.
	\item[---] Реализовано мультиплексирование -- каждый процесс или поток должен отдавать данные по нескольким сетевым соединениям.
	\item[---] Сервер по умолчанию возвращает HTML-страницу на выбранную тему с CSS-стилем.
	\item[---] Реализована запись информации о событиях в журнал (лог).
	\item[---] Учтены минимальные требования к безопасности серверов статического содержимого.
\end{itemize}
